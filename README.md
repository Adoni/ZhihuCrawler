Zhihu Crawler
====
[![license][badge-license]][license]

A web spider to crawl networks, user profiling and user generated content on [Zhihu](https://www.zhihu.com)

##Dependency
* Requests
* Zhihu-OAuth
* Redit
* MongoDB
